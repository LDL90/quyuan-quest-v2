# 🐉 屈原端午亲子公园探秘

> 剧本杀风格 · 亲子互动 · 公园寻游 · 微信扫码即用

## 🎮 项目介绍

这是一个专为端午节设计的亲子户外互动网页游戏，采用**剧本杀沉浸式体验**，让孩子们在公园里边玩边学屈原文化。

### ✨ 特色亮点

- 🎭 **剧本杀体验**：穿越楚国，与NPC对话，收集线索碎片
- 🎨 **水墨国风**：金色、朱红、翠玉配色，Canvas粒子背景
- 📱 **微信扫码**：无需下载，手机扫码即玩
- 🏆 **通关证书**：通关后可打印专属证书
- 📊 **数据管理**：管理员面板统计参与数据

## 🚀 使用方法

1. **扫码开始**：用微信扫描下方二维码
2. **输入信息**：填写孩子姓名和家长手机号
3. **闯关探秘**：跟随NPC对话，回答问题
4. **领取证书**：通关后打印专属证书

## 🔐 管理员功能

- 密码：`8023`
- 查看统计数据（人数、平均分、满分）
- 导出CSV数据
- 清空历史记录
